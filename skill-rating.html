How to send stats

Use a clean screenshot of one player row from Tracker. The screenshot is read in the browser, then you can correct every detected value before sending the report.

Step 01Open Tracker

Go to tracker.gg, open your profile, then choose the last match you played.

Step 02Find your row

Find yourself in the match table and screenshot only the row with your stats.

Step 03Send this shape

The best screenshot shows only the row in this order: ACS, K, D, A, +/-, K/D, DDA, ADR, HS%, KAST, FK, FD, MK.

Tip: you can upload the file or paste a screenshot directly with Ctrl+V in the Submit Match window. If OCR reads something wrong, correct the value before submitting.
